    2. **`SignatureVerificationResult` enum**:

       ```rust
       pub enum SignatureVerificationResult {
           Valid { fingerprint: String },
           Invalid { reason: String },
           Untrusted { fingerprint: String },
           NoSignature,
       }
       ```

    3. **`verify_git_tag_signature(tag_name: &str, repo_dir: &Path, trust_store: &TrustStore) -> Result<SignatureVerificationResult>`**:
       - Uses `std::process::Command` to `git -C <repo> verify-tag --raw <tag>` and captures output
       - Parses the `--raw` output to extract the signature fingerprint
       - Checks if the fingerprint is in the trust store
       - Returns `Valid`, `Invalid`, or `Untrusted` based on verification outcome
       
       Rationale for using system git (not libgit2 binding): Git's built-in tag verification handles the complex OpenPGP packet parsing, key lookup, and trust chain verification. We just need to verify the outcome and check the author's key fingerprint against our trust store. This avoids duplicating git's own GPG integration.

    4. **`verify_manifest_signature(manifest: &SkillManifest, tag_name: &str, repo_dir: &Path, trust_store: &TrustStore) -> Result<SignatureVerificationResult>`**:
       - Wraps `verify_git_tag_signature`
       - Cross-references the signature fingerprint against `manifest.gpg.fingerprint`
       - If manifest declares a fingerprint and it doesn't match the actual signer → `Invalid`
       - If no GPG config in manifest → `NoSignature` (not an error, caller decides policy)

    **Error handling:**
    - `VerifyError` enum with `thiserror`:
      - `GitError(String)` — git command failed
      - `KeyNotFound(String)` — cert not found
      - `CertParse(String)` — cert parsing failure
      - `SignatureInvalid(String)` — bad signature
    - Wrap git command output in `anyhow::Context`

    **Conventions**:
    - Logging: `tracing::info!("[gpg] verifying {tag} against {fingerprint}")`
    - Cert file naming: `{fingerprint}.asc` (uppercase hex, no spaces/colons)
    - Cert storage format: ASCII-armored OpenPGP transferable public key

    **Step B — Implement the full `verify_git_tag_signature` function:**

    The git command integration pattern:
    ```rust
    pub fn verify_git_tag_signature(
        tag_name: &str,
        repo_dir: &Path,
        trust_store: &TrustStore,
    ) -> Result<SignatureVerificationResult, VerifyError> {
        let output = std::process::Command::new("git")
            .arg("-C")
            .arg(repo_dir)
            .arg("verify-tag")
            .arg("--raw")
            .arg(tag_name)
            .output()
            .map_err(|e| VerifyError::GitError(format!("failed to execute git: {e}")))?;
        
        // git verify-tag --raw outputs to stderr
        let stderr = String::from_utf8_lossy(&output.stderr);
        let stdout = String::from_utf8_lossy(&output.stdout);
        
        // If git exits 0, signature is cryptographically valid.
        // Parse stderr for the fingerprint.
        if output.status.success() {
            // Example --raw output:
            // "gpg: Signature made ...\ngpg:                using RSA key A1B2C3D4..."
            // "gpg: Good signature from \"Author Name <author@example.com>\""
            // Extract the key fingerprint from the stderr lines
            // Parse "using XXX key <FINGERPRINT>" pattern
            let fingerprint = extract_fingerprint(&stderr)
                .ok_or_else(|| VerifyError::SignatureInvalid(
                    "could not extract fingerprint from git output".into()
                ))?;
            
            // Check if fingerprint is in trust store
            if trust_store.find_cert(&fingerprint)?.is_some() {
                Ok(SignatureVerificationResult::Valid { fingerprint })
            } else {
                Ok(SignatureVerificationResult::Untrusted { fingerprint })
            }
        } else {
            let reason = stderr.lines()
                .find(|l| l.contains("BAD signature") || l.contains("Can't check signature"))
                .unwrap_or("signature verification failed")
                .to_string();
            Ok(SignatureVerificationResult::Invalid { reason })
        }
    }
    ```
