    Create `src/openhuman/skills/wasm_install.rs` implementing the full install workflow:

    1. **`GitSkillInstaller` struct** — orchestrates install:
       ```rust
       pub struct GitSkillInstaller {
           pub store: SkillsStore,
           pub trust_store: TrustStore,
           pub wasm_engine: Arc<WasmEngine>,
       }
       ```

    2. **`install_skill(git_url: &str, trust_store: &TrustStore, store: &mut SkillsStore) -> Result<InstallOutcome>`** — the full pipeline:
       
       **Step 1 — Clone repository:**
       - Parse git URL, derive local clone path: `~/.openhuman/skills/<name>/`
       - `git clone <url> <path> --depth 1` (shallow clone for speed)
       - Validate path does not escape skills root (same canonical path guard used by existing `uninstall_skill`)
       - Timeout: 120 seconds via `std::process::Command::timeout` or async equivalent
       
       **Step 2 — Read and validate manifest:**
       - Read `<clone-path>/dadou-skill.yaml`
       - Call `parse_manifest()` — reject if invalid
       
       **Step 3 — Determine latest signed tag:**
       - `git -C <path> tag --list --sort=-version:refname` to get tags
       - Pick latest semver tag, or use `HEAD` if no tags
       
       **Step 4 — GPG verification (if gpg.fingerprint in manifest):**
       - Call `verify_git_tag_signature(tag, path, trust_store)`
       - If result is Untrusted, Invalid, or NoSignature when GPG is required → abort
       - (If manifest has no gpg.fingerprint → warn but continue per user policy)
       
       **Step 5 — Checkout the verified tag:**
       - `git -C <path> checkout <tag>` if a tag was verified
       
       **Step 6 — Static analysis:**
       - Call `scan_skill(path, &manifest.permissions)`
       - If verdict is Block → abort with findings details
       - If verdict is Warn → log warning, continue (user can review later)
       
       **Step 7 — Compile WASM (if source provided, not pre-compiled):**
       - If `dadou-skill.yaml` references a `.wasm` file directly, use it
       - If source-only (future enhancement), skip for v1; require pre-compiled .wasm
       
       **Step 8 — Register in store:**
       ```rust
       let installed = InstalledSkill {
           name: manifest.name.clone(),
           version: manifest.version.clone(),
           commit_hash: resolved_commit_hash,
           enabled: true,
           gpg_fingerprint: verified_fingerprint,
           installed_at: chrono::Utc::now().to_rfc3339(),
           last_audit_at: Some(chrono::Utc::now().to_rfc3339()),
           audit_result: Some(if analysis.verdict == AnalysisVerdict::Block { "fail" } else { "pass" }),
       };
       store.upsert(installed)?;
       ```

    3. **`InstallOutcome` struct**:
       ```rust
       pub struct InstallOutcome {
           pub name: String,
           pub version: String,
           pub gpg_status: String,     // "verified" | "untrusted" | "no_signature"
           pub analysis_verdict: AnalysisVerdict,
           pub findings_count: usize,
           pub path: PathBuf,
       }
       ```

    4. **`update_skill(name: &str, store: &mut SkillsStore) -> Result<InstallOutcome>`**:
       - Fetch latest from existing git remote
       - Re-verify GPG on latest tag
       - Re-run static analysis
       - Update store entry

    5. **`audit_skill(name: &str, store: &mut SkillsStore) -> Result<AuditOutcome>`**:
       - Load skill from store
       - Find installed path
       - Run static analysis only
       - Update `audit_result` and `last_audit_at` in store
       - Return findings

    6. **`remove_skill(name: &str, store: &mut SkillsStore) -> Result<RemoveOutcome>`**:
       - Remove from store
       - Remove skill directory `~/.openhuman/skills/<name>/`
       - Return success

    **Error handling**:
    - `InstallError` enum with variants: GitError, ManifestError, GpgError, AnalysisBlocked, StoreError
    - Use `anyhow::Context` for error wrapping
    - Detailed logging at each step: `tracing::info!("[skill-install] step X: {detail}")`
    
    **Conventions**:
    - Git URL validation: must start with `https://` or `git@`, reject `file://` and `ssh://` without host
    - Shallow clone: `git clone --depth 1` for speed
    - Path safety: canonicalize clone path, verify it stays under `~/.openhuman/skills/`

    ```rust
    fn run_dadou_skill_command(args: &[String]) -> Result<()> {
        use crate::openhuman::skills::wasm_install::*;
        
        if args.is_empty() || is_help(&args[0]) {
            println!("Usage: dadou skill <subcommand> [options]");
            println!();
            println!("Subcommands:");
            println!("  install <git-url>    Install a skill from a Git repository");
            println!("  update <name>        Update an installed skill to latest version");
            println!("  audit <name>         Run static analysis on an installed skill");
            println!("  remove <name>        Uninstall a skill");
            println!("  list                 List installed skills with their state");
            println!("  trust-author <fp>    Add a GPG public key fingerprint to trusted authors");
            println!();
            println!("Examples:");
            println!("  dadou skill install https://github.com/author/skill-repo.git");
            println!("  dadou skill audit my-skill");
            println!("  dadou skill list");
            return Ok(());
        }

        match args[0].as_str() {
            "install" => {
                if args.len() < 2 {
                    return Err(anyhow::anyhow!("Usage: dadou skill install <git-url>"));
                }
                // Use tokio runtime for async install
                let rt = tokio::runtime::Runtime::new()?;
                rt.block_on(async {
                    // Parse git URL
                    let url = &args[1];
                    let store = SkillsStore::load()?;
                    let trust_store = TrustStore::load()?;
                    let engine = Arc::new(WasmEngine::new()?);
                    let installer = GitSkillInstaller { store, trust_store, wasm_engine: engine };
                    let outcome = installer.install_skill(url).await?;
                    // Print outcome in a structured format (JSON for machine parsing, human-readable summary)
                    println!("Installed skill: {} v{}", outcome.name, outcome.version);
                    println!("  GPG: {}", outcome.gpg_status);
                    println!("  Static analysis: {:?}", outcome.analysis_verdict);
                    Ok(())
                })
            }
            "update" => {
                if args.len() < 2 { return Err(anyhow::anyhow!("Usage: dadou skill update <name>")); }
                // Similar async pattern
                todo!("Implement in execution")
            }
            "audit" => {
                if args.len() < 2 { return Err(anyhow::anyhow!("Usage: dadou skill audit <name>")); }
                let name = &args[1];
                let mut store = SkillsStore::load()?;
                let outcome = audit_skill(name, &mut store)?;
                println!("Audit result for {}: {:?}", name, outcome.verdict);
                for finding in &outcome.findings {
                    println!("  [{:?}] {}:{} — {}", finding.severity, finding.file, finding.line, finding.pattern);
                }
                Ok(())
            }
            "remove" => {
                if args.len() < 2 { return Err(anyhow::anyhow!("Usage: dadou skill remove <name>")); }
                let mut store = SkillsStore::load()?;
                let outcome = remove_skill(&args[1], &mut store)?;
                println!("Removed skill: {}", outcome.name);
                Ok(())
            }
            "list" => {
                let store = SkillsStore::load()?;
                let skills = store.list();
                if skills.is_empty() {
                    println!("No skills installed.");
                } else {
                    println!("Installed skills:");
                    for skill in skills {
                        let status = if skill.enabled { "enabled" } else { "disabled" };
                        println!("  {} v{} [{}]", skill.name, skill.version, status);
                        if let Some(audit) = &skill.audit_result {
                            println!("    Audit: {}", audit);
                        }
                    }
                }
                Ok(())
            }
            "trust-author" => {
                if args.len() < 2 { return Err(anyhow::anyhow!("Usage: dadou skill trust-author <fingerprint>")); }
                // Future: import GPG cert by fingerprint
                println!("trust-author: not yet implemented (add cert file to ~/.openhuman/skills/certs/)");
                Ok(())
            }
            _ => {
                println!("Unknown skill subcommand: {}", args[0]);
                println!("Run 'dadou skill --help' for usage.");
                Ok(())
            }
        }
    }
    ```
