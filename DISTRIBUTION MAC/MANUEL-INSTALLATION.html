🧠 DADOU — Manuel d'installation

macOS 14+ (Sonoma / Sequoia) · Déploiement local · Rust + Tauri v2

Prêt en 15 minutes. Ce guide couvre l'installation complète : prérequis, compilation, configuration et lancement.

1. Prérequis

ComposantVersion minVérification
macOS14 (Sonoma)sw_vers -productVersion
Xcode CLI Tools—xcode-select -p
Homebrew—brew --version
Rust1.93rustc --version
Node.js≥24node --version
pnpm≥10.10pnpm --version
Git—git --version
Ollama (optionnel)—ollama --version

Vérification automatique

cd "DISTRIBUTION MAC"
bash prerequis-mac.sh --install

Ce script vérifie tous les prérequis et installe les composants manquants via Homebrew.

2. Installation rapide

Installation automatisée

cd "DISTRIBUTION MAC"
bash install.sh

Ce script exécute les 5 étapes : prérequis → clonage → build → configuration → vérification.

Mode hors-ligne (sans backend cloud)

bash install.sh --offline

Configure DADOU pour fonctionner sans connexion au backend cloud. L'IA locale (Ollama) est nécessaire.

Installation manuelle

# 1. Cloner
git clone https://github.com/macao100/openhuman.git ~/dadou
cd ~/dadou

# 2. Submodules Tauri
git submodule update --init --recursive

# 3. Dépendances
pnpm install --frozen-lockfile

# 4. Compiler le core Rust (release)
cargo build --release --bin openhuman-core

# 5. Compiler le frontend
pnpm build

# 6. Configurer l'environnement
cp .env.example .env
cp app/.env.example app/.env.local
mkdir -p ~/.openhuman

3. Configuration post-installation

Variables d'environnement essentielles

VariableDéfautDescription
OPENHUMAN_OFFLINE_MODEfalsetrue = sans backend, IA locale uniquement
OPENHUMAN_MODELautoModèle LLM par défaut (ex: llama3.2)
OPENHUMAN_CORE_PORT7788Port du serveur JSON-RPC
OPENHUMAN_WORKSPACE~/.openhumanRépertoire de travail
RUST_LOGinfoNiveau de log (debug, info, warn, error)

Éditer .env (racine) et app/.env.local pour personnaliser. Voir le template complet dans .env.example.

4. Lancement

Mode desktop (Tauri)

cd ~/dadou
pnpm dev:app

Lance l'application desktop complète avec interface graphique.

Mode serveur (JSON-RPC headless)

cd ~/dadou
cargo run --release -- serve

Démarre le serveur JSON-RPC sur http://127.0.0.1:7788/rpc.

Mode CLI

cd ~/dadou
cargo run --release -- help

5. Vérification

Tests unitaires : cargo test --lib -p dadou
ESLint : cd app && pnpm lint
TypeScript : pnpm compile
Format : cargo fmt --check
Note : Sur macOS, les tests Guardian N2 (float) et les tests de sandbox (cwd_jail) passent correctement. Les tests WASM/Python sont ignorés (runtime retiré). Attendu : 0 échec.
