3Compilation et installation

Compilation du core Rust et du frontend Tauri.

Étape 3a — Dépendances frontend

# Installation des packages npm pnpm install

Étape 3b — Compilation Rust

# Vérification du core Rust cargo check --manifest-path Cargo.toml

Étape 3c — Build production

# Build complet (core + UI) cargo build --manifest-path Cargo.toml --bin openhuman-core --release pnpm build
4Premier lancement

Démarrez DADOU et ouvrez le dashboard.

Lancer le core

.\target\release\openhuman-core.exe serve

Dashboard
